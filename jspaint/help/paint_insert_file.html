Per inserire una bitmap nell'immagine corrente

  1. Nella cassetta degli strumenti, fare clic su e poi trascina il puntatore per definire un'area per mettere la bitmap.
  2. Sul Modificare Menu, clicca Incolla da.
  3. Individua e fai doppio clic sul file Bitmap che si desidera inserire.
  4. Trascina la bitmap dove lo desideri, quindi fai clic fuori dalla selezione.
