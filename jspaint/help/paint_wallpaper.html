Per usare un'immagine come sfondo del desktop

  1. Sul File Menu, fare clic su uno dei seguenti comandi:
    • Set come sfondo (piastrellato) Copre lo schermo con ripetizioni della tua foto.
    • Set come sfondo (centrato) Posiziona la tua foto al centro dello schermo.
  2. Salva l'immagine.
  3. Vai nelle impostazioni del sistema per scegliere lo sfondo.

Note

  • JS Paint non può impostare direttamente il tuo sfondo, ma può generare un'immagine piastrellata delle dimensioni del desktop.
  • The Set come sfondo (centrato) L'opzione non utilizza attualmente la dimensione del desktop, salva solo una copia dell'immagine. È necessario assicurarsi che nelle impostazioni del sistema scelgano di centrarre l'immagine.
