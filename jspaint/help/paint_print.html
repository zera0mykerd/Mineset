Per stampare un'immagine

Sul File Menu, clicca Stampa.

Note

  • Puoi vedere come sarà l'immagine stampata prima di stampare facendo clic File menu e quindi fare clic Anteprima di stampa.
  • È possibile impostare i margini o modificare l'orientamento facendo clic File menu e quindi fare clic Impostazione della pagina.
