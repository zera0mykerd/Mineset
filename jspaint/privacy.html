MSPaint JS
politica sulla riservatezza

JS Paint è un remake pixel-perfetto di Microsoft Paint che viene eseguito nel browser.

La presente informativa sulla privacy si applica al sito Web JS Paint all'indirizzo jspaint.app

Dati personali

JS Paint non ti traccia né vende i tuoi dati.

Servizi di terze parti

Nella maggior parte dei casi JS Paint non invia alcuna informazione a un server.

Ci sono alcune funzionalità che comunicano con un server:

  • File > Carica su Imgur richiede naturalmente l'invio dell'immagine a Imgur per ospitare.
  • Se incolli l'URL di un'immagine, l'app recupererà l'immagine tramite l'URL, o se non funziona, recupera tramite a CORS Ovunque servizio proxy o, se non funziona, provare a recuperarlo tramite il file Internet Archive Wayback Machine
  • Nella modalità Riconoscimento vocale (Extra > Riconoscimento vocale), se dici ad esempio "disegna un gatto", proverà a eseguire una ricerca anonima di immagini Bing per clipart "gatto", e poi animarlo disegnandolo.
  • Modalità multiutente (Extra > Multiutente), destinato alla collaborazione online, utilizza Firebase per la comunicazione in tempo reale.

Alcune di queste funzionalità potrebbero essere inattive a causa di cambiamenti nei servizi su cui fanno affidamento. Potranno essere sostituiti o rimossi in futuro, per semplificare l'informativa sulla privacy.

Archiviazione locale

JS Paint utilizza l'archiviazione locale per salvare il lavoro in corso.

È possibile visualizzare ed eliminare le immagini memorizzate con File > Gestisci spazio di archiviazione

Consideralo come un backup, nel caso in cui chiudi la scheda per errore o il tuo computer si blocchi. Non fare affidamento su di esso come unico backup.

Al momento non esiste alcuna opzione per impedire all'app di salvare un backup, tuttavia puoi utilizzare la modalità di navigazione privata del tuo browser per impedire che i dati vengano persistenti.

Se l'URL inizia con https://jspaint.app/#local:, è una sessione locale, privata per te.

Sessioni collaborative

Le sessioni multiutente sono attualmente pubbliche. Chiunque potrebbe trovarli e visualizzarli, modificarli o eliminarli.

Le sessioni collaborative private non sono ancora supportate.

Se l'URL inizia con https://jspaint.app/#session:, è una sessione multiutente.
